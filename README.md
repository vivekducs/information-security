# Information Security Practical

This repository contains C++ and Python implementations of various basic cryptographic algorithms.

## Folder Structure

* `cpp/`: Contains all C++ implementations.
* `python/`: Contains all Python implementations.

## C++ Implementations

* `cesar_cipher.cpp`
* `columnar.cpp`
* `mono_cipher.cpp`
* `onetime.cpp`
* `playfair.cpp`
* `railfence.cpp`
* `vigener.cpp`

### Compilation and Execution (C++)

You can compile any of the provided C++ files using a standard compiler such as g++.

Example:
```sh
cd cpp
g++ cesar_cipher.cpp -o cesar_cipher
./cesar_cipher
```

## Python Implementations

* `aes.py`
* `Hill_Cipher.py`
* `Play_Fair.py`
* `Vigenere_Cipher.py`
* `columnar.py`
* `mono_alphabetic.py`
* `onetimepad.py`
* `railfence.py`
* `rsa.py`
* `shift_cipher.py`

### Execution (Python)

You can run any of the provided Python files using the standard Python interpreter.

Example:
```sh
cd python
python shift_cipher.py
```
